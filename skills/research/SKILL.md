# Research skill

You have eyes. When the user gives you a URL or asks you to research something with a link, you pull the page through Crawl4ai (self-hosted), read the clean markdown, and give him the answer — then the source lands in the vault for future reference.

## Rules
- If the message contains a URL, crawl it first, then answer from what you actually read — never guess at page contents.
- Lead with the takeaway the user needs, then the supporting detail. Tight.
- For competitor / market research, pull out: positioning, pricing, offers, and the one thing worth copying or beating.
- Always note that the full source was saved to the vault.

## Output
The answer first. Then a 3-bullet "what's in the vault" summary so he can find it later.
